# Install instructions

#### Docker Services SQLite 3, Postgres 12, MySQL 8
`$ docker-compose up -d`

####  install Node.js dependies
`$ npm install` \
If you have build proble with SQLite 3 module on the Wondows run \
`$ npm config set msvs_version 2015`
